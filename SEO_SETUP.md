# SEO Setup Guide

This document outlines the SEO configuration for the Sortask website.

## ‚úÖ Completed Setup

### 1. Sitemap (`/sitemap.xml`)
- Automatically generated by Next.js from `app/sitemap.ts`
- Includes all main pages with proper priorities and change frequencies
- Accessible at: `https://sortask.com/sitemap.xml`

### 2. Robots.txt (`/robots.txt`)
- Automatically generated by Next.js from `app/robots.ts`
- Allows all search engines to crawl the site
- Disallows `/api/` and `/_next/` directories
- References the sitemap location
- Accessible at: `https://sortask.com/robots.txt`

### 3. Metadata & SEO
- **Root Layout**: Comprehensive metadata including Open Graph, Twitter Cards, and keywords
- **Page-specific metadata**: Each main page has its own metadata for better SEO
- **Structured Data**: JSON-LD schema for Organization and WebSite

### 4. Structured Data (JSON-LD)
- Organization schema with company information
- Website schema with search functionality
- Automatically included in all pages

## üîß Configuration Needed

### Update Domain Name
You need to update the domain name in the following files:

1. **`app/sitemap.ts`** - Line 4:
   ```typescript
   const baseUrl = 'https://sortask.com' // Change to your actual domain
   ```

2. **`app/robots.ts`** - Line 4:
   ```typescript
   const baseUrl = 'https://sortask.com' // Change to your actual domain
   ```

3. **`app/layout.tsx`** - Line 20:
   ```typescript
   metadataBase: new URL('https://sortask.com'), // Change to your actual domain
   ```

### Add Images
1. **Open Graph Image**: Add `/public/og-image.jpg` (1200x630px recommended)
2. **Twitter Image**: Add `/public/twitter-image.jpg` (1200x630px recommended)
3. **Logo**: Add `/public/logo.png` for structured data

### Google Search Console
1. Add your website to Google Search Console: https://search.google.com/search-console
2. Verify ownership using one of these methods:
   - HTML file upload
   - HTML tag (add verification code to `app/layout.tsx` metadata.verification.google)
   - Domain name provider
   - Google Analytics

### Google Analytics (Optional but Recommended)
1. Create a Google Analytics 4 property
2. Add the tracking code to `app/layout.tsx` or use `next/script`

### Social Media Links
Update `app/components/StructuredData.tsx` with your actual social media profiles:
```typescript
sameAs: [
  'https://twitter.com/sortask',
  'https://linkedin.com/company/sortask',
  'https://github.com/sortask',
],
```
